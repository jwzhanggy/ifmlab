    Large-Scale Multi-Label Learning with Incomplete Label Assignments

    Xiangnan Kong, Zhaoming Wu, Li-Jia Li, Ruofei Zhang, Philip S. Yu, Hang Wu and Wei Fan
    SIAM International Conference on Data Mining (SDM)
    We study the problem of large-scale multi-label learning, where each instance can be assigned with multiple labels simultaneously and the label assignments for training instances can be incomplete. We propose an approach, called MPU, based upon positive and unlabeled stochastic gradient descent and stacked models. Our MPU method can effectively and efficiently consider missing labels and label correlations simultaneously, and is very scalable, that has linear time complexities over the size of the data. We demonstrate the empirical efficacy of MPU on two real-world multi-label data sets and show that MPU method can effectively learn from multi-label datasets with incomplete label assignments.

    Multi-Label Feature Selection for Graph Classification

    Xiangnan Kong and Philip S. Yu
    IEEE International Conference on Data Mining (ICDM) (Full paper, acceptance 9.03%)
    Most of the graph objects in graph classification datasets can have multiple functions simultaneously. In this paper, we study the problem of multi-label feature selection for graph classification, where each instance is represented as graph object and each graph object can be assigned with multiple labels simultaneously. We propose an approach, called gMLC, to efficiently search for optimal subgraph features for graph objects with multiple labels. Then we propose a branch-and-bound algorithm to efficiently search for optimal subgraph features by pruning the subgraph search space using multiple labels.

  • Semi-supervised Feature Selection for Graph Classification

    Xiangnan Kong and Philip S. Yu
    ACM SIGKDD Conference on Knowledge Discovery and Data Mining (KDD) (Full paper, acceptance 13.3%)
    Very few labeled examples are available in most graph classification datasets. In this paper, we study the problem of semi-supervised feature selection for graph classification, where each instance is represented as graph object and only a small fraction of the label assignments for training instances is available. We propose an approach, called gSSL, to efficiently search for optimal subgraph features with labeled and unlabeled graphs. Then we propose a branch-and-bound algorithm to efficiently search for optimal subgraph features by pruning the subgraph search space using both labeled and unlabeled graphs.
